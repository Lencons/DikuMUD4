<pre>integer spellindex(s&#160;: string)
   s  : The abbreviated or full spell name
</pre>
<pre>   Returns: -1 if no such spell, otherwise it returns the spell index
            for that spell.
</pre>
<pre>   Example:
      s&#160;:= "cu li wo"; /* cure light wounds */
      i&#160;:= spellindex(s);
</pre>
